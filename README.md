**Azure Virtual Network & VM Deployment**
### My Goal
To build a cloud environment by setting up a custom Virtual Network and an Ubuntu Linux Virtual Machine on Microsoft Azure.

### What I Did
 **Created a VNet:** Built a private network space in the Sweden Central region.
 **Configured a Subnet:** Set up a `Public_Subnet` (10.0.0.0/24).
 **Deployed a VM:** Launched an Ubuntu server and confirmed it is "Running."
